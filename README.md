## üíª Getting Started

This is a [Next.js](https://nextjs.org) project. To run the portfolio locally on your machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Ayush-kathil/ayush-p.git](https://github.com/Ayush-kathil/ayush-p.git)
    cd ayush-p
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://ayushgupta3.vercel.app) to see the result.
